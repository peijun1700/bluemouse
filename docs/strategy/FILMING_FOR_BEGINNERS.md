# 🎬 超簡單拍攝法：連小白都會 (Zero-Tech Method)

**核心概念**: 您不需要真的操作 BlueMouse，只需要「演」出那個效果就好。
**工具**: 手機 + 這三張封面圖 + 您的聲音

---

## 方法 1: 純圖片 + 旁白 (最簡單)

**您需要的素材**:
1. 三張封面圖 (我已經生成好了，在下方)
2. 手機的「照片」App
3. 剪映 App (免費)

**步驟**:
1. **打開剪映**，選「照片電影」模式
2. **導入三張圖**:
   - `SAVED MY JOB` (刪庫危機)
   - `SAVED $5K` (AWS 帳單)
   - `LEAK BLOCKED` (金鑰外洩)
3. **每張圖停留 3-4 秒**
4. **加上您的旁白** (照著下面的稿念):

### 旁白稿 (中文版)

**第一張圖 (SAVED MY JOB)**:
> "視角：AI 差點刪了我的正式資料庫... 但 BlueMouse 在最後一秒攔截了它。這個藍色視窗，救了我的職業生涯。"

**第二張圖 (SAVED $5K)**:
> "我從 ChatGPT 複製了一段代碼，裡面藏著無限迴圈。如果部署到 AWS，我的帳單會爆炸。BlueMouse 在我按下 Deploy 之前就抓到了。"

**第三張圖 (LEAK BLOCKED)**:
> "我差點把 API Key 推到公開的 GitHub。BlueMouse 的邏輯防火牆攔下了這次資安災難。連結在首頁，免費下載。"

### 旁白稿 (英文版)

**Image 1 (SAVED MY JOB)**:
> "POV: AI almost deleted my production database... but BlueMouse stopped it at the last second. This blue modal just saved my career."

**Image 2 (SAVED $5K)**:
> "I copied code from ChatGPT with a hidden infinite loop. If deployed to AWS, my bill would explode. BlueMouse caught it before I hit deploy."

**Image 3 (LEAK BLOCKED)**:

## 方法 2: 螢幕截圖 + Ken Burns 效果 (進階一點點)

如果您想要「看起來像真的在操作電腦」:

1. **截圖您的 VS Code 畫面** (隨便打開一個 Python 檔案就好)
2. **在剪映裡導入截圖**
3. **套用「Ken Burns」特效** (鏡頭會自動放大/移動，看起來像在操作)
4. **疊加我們的封面圖** (在關鍵時刻彈出)
